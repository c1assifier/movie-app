# VK Test App

## Описание

**VK Test App** — это SPA-приложение для просмотра информации о фильмах, разработанное в рамках тестового задания.
В приложении есть каталог фильмов с фильтрами, детальная страница фильма, избранное с сохранением после перезагрузки и режим сравнения двух фильмов.

---

## Демо

[Посмотреть live demo](https://vk-test-app-seven.vercel.app/)

> Для live demo в проекте предусмотрен fallback на локальные mock-данные.
> Это сделано из-за ограничений бесплатного тарифа API: приложение использует ПоискКино API (`https://poiskkino.dev`), а запросы идут к `https://api.poiskkino.dev/v1.4`.
> На демо-тарифе доступно только 200 запросов в сутки, поэтому при отсутствии ключа или исчерпанном лимите приложение продолжает работать на mock-данных без падения основных сценариев.
> Если вы используете собственный API key, рекомендуется следить за доступным лимитом запросов!

## Функциональность

- Каталог фильмов с загрузкой данных с API
- Фильтрация по жанрам, рейтингу и году выпуска
- Синхронизация фильтров с search params
- Восстановление состояния фильтров по прямой ссылке
- Бесконечный скролл с подгрузкой фильмов по 50 записей
- Детальная страница фильма
- Избранное с сохранением в localStorage
- Подтверждение добавления фильма в избранное через модальное окно
- Сравнение двух фильмов

## Запуск проекта

1. Клонировать репозиторий

```bash
git clone https://github.com/c1assifier/movie-app.git
```

2. Перейти в директорию проекта

```bash
cd movie-app
```

3. Создать локальный `.env` на основе `.env.default`

```bash
cp .env.default .env
```

4. Указать API key в `.env`

```env
POISKKINO_API_KEY=вставить_свой_ключ
```

5. Установить зависимости

```bash
npm install
```

6. Запустить проект

```bash
npm run dev
```
